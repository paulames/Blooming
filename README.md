# Blooming
Aplicación para el seguimiento emocional de los alumnos en el entorno escolar de primaria y principios de secundaria.

## Tecnologías Utilizadas

### **Lenguajes de Programación**
- **JavaScript (JS)** 
- **TypeScript (TS)**

### **Lenguajes de Marcado y Estilo**
- **HTML5**
- **CSS3**
- **Bootstrap**

### **Frameworks**
- **Angular**
